1. @import : gọi các file .scss khác
Trong 1 dự án phân chia thành nhiều scss để dễ quản lý và mở rộng.
Khi cần chỉnh sửa chỉ cần tìm đúng file.
Tạo 1 file định nghĩa mã màu :
_ Tạo variables.scss :
$navbar-height: 77px
$sidebar-width: 164px;
$font-bold: 700;
_ Tạo colors.scss :
$primary-color : #0076ff;  // Màu chính, màu cơ bản
$secondary-color : #f6f9ff; // Màu phụ
$border-color : #edf0ff;
$text-color: #000;
$background-color : #fff;
... navbar, label, success-color, failure-color

Sử dụng tại 1 file style.scss

@import "colors";
p{
   color : $primary-color;
   background-color : $background-color;
}

p:hover{
   border-color : $border-color
}

=> Bổ sung _variables, _colors để không bị biên dịch
=> Giống như private vậy. Gọi là partial


2. @media : liên quan reponsive :D
@media all and (min-width : 1200){
   body{
     background-color: red;
   }
}

@media all and (min-width : 768) and (max-width: 1200){
   body{
     background-color: green;
   }
}


@media all and (max-width : 768){
   body{
     background-color: yellow;
   }
}

@media all and ( orientation: landscape ){
   body{
     background-color: blue;
   }
}

3. @extend :

.message{
   font-size : 24px;
   padding: 10px;
}

.success{
   @extend .message;
   background-color: green;
   border: solid 1px green;
   color : red;
}

.failure{
   @extend .message;
   background-color: red;
   border: solid 1px red;
   color : yellow;
}
=> Tùy vào cách viết hợp lý.

đa kế thừa :

.important {
  font-weight : bold;
  text-decoration : underline
}

.message-success-important{
   @extend .message, .success, .important // hoặc viết xuống dòng
}

=> Multiple Extend :

.error {
  border: 1px #f00;
  background-color: #fdd;
}
.attention {
  font-size: 3em;
  background-color: #ff0;
}
.seriousError {
  @extend .error;
  @extend .attention;
  border-width: 3px;
}



=> Chainning Extend

.error {
  border: 1px #f00;
  background-color: #fdd;
}
.seriousError {
  @extend .error;
  border-width: 3px;
}
.criticalError {
  @extend .seriousError;
  position: fixed;
  top: 10%;
  bottom: 10%;
  left: 10%;
  right: 10%;
}

_ Tạo phần định dạng như không muốn gọi biên dịch ra css :
%important{
  font-weight : bold;
  text-decoration : underline
}
.message-success-important{
   @extend %important // hoặc viết xuống dòng
}
=> Chỗ nào có %important đỗ class đang cần định dạng vào.
