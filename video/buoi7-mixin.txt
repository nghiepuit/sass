1. Mixin : ( giống như extends - có hỗ trợ truyền tham số )

$background-color : "red"
@mixin button{
  background-color : $background-color;
  color : yellow;
  border-radius : 2px;
  border : none;
  transition: 0.5s;
  &:hover, &:active, &:focus{
    background-color: lighten($background-color, 70%);
    color: $text-color;
  }
}

@include button
=> Chỉ include mới được sinh ra

_ Trường hợp khác :
.container {
   @include button
}

=> .container button{
   .....
}


_ Kết hợp Mixin :

@mixin bg-text{
   background-color : blue;
}

@mixin title{
   color : yellow
}

@mixin compound{
   @include bg-text;
   @include header-text;
}

h1{
  @include compound
}
