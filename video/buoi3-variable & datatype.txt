2. Phạm vi :
$primary-color : red;
$text-size: 20px;

div.container{
   $secondary-color : blue; // chỉ có phạm vi bên trong thẻ div có class container
   p{
	font: {
	   size: $text-size;
        }
	color : $secondary-color // hoặc $primary-color 
   }  
}

div.footer{
  p{
	color : $primary-color 
  }
}

=> Mở rộng : Để có thể gọi 1 biến bên trong 1 block khác. VD như thẻ p trong div.footer gọi phần định dạng $secondary-color => dùng !global phía sau giá trị thuộc tính

div.container{
   $secondary-color : blue !global; // chỉ có phạm vi bên trong thẻ div có class container
   p{
	font: {
	   size: $text-size;
        }
	color : $secondary-color // hoặc $primary-color 
   }  
}

div.footer{
  p{
	color : $secondary-color
  }
}
